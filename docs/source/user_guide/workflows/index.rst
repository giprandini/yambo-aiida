Avaliable calculation plugin
-----------------------------

Description
^^^^^^^^^^^
`Yambo`_  is a  code for Many-Body calculations in solid state and molecular physics  `available online`_.
Documentation of the code and its internal details can also be found in the `Yambo Documentation`_.

.. _Yambo: http://www.yambo-code.org/
.. _available online: https://github.com/yambo-code/yambo/wiki/Releases-%28tar.gz-format%29
.. _Yambo Documentation: http://www.yambo-code.org/theory/index.php

Versions 4.2.0 and newer of yambo  are fully supported.

* `yambo`: GW  quasiparticle computations.
